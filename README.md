# file2file.me

Free online file converter for documents. Convert CSV, Excel, PDF, and Word files instantly in your browser.

## Features

- **CSV to Excel** - Convert CSV files to Excel spreadsheets
- **Excel to CSV** - Export Excel data to CSV format
- **PDF to Word** - Convert PDF documents to editable Word files
- **Word to PDF** - Create PDF from Word documents
- **Excel to PDF** - Convert spreadsheets to PDF
- **CSV to PDF** - Generate PDF from CSV data

## Why file2file.me?

- **Free** - No hidden costs or subscriptions
- **Fast** - Instant file conversion in your browser
- **Secure** - Files processed locally, no upload to servers
- **No Registration** - Start converting immediately
- **Cross-platform** - Works on any device with a browser

## Supported Formats

| Input Format | Output Formats |
|--------------|----------------|
| CSV | Excel, PDF |
| Excel (XLSX, XLS) | CSV, PDF |
| PDF | Word (DOCX) |
| Word (DOCX, DOC) | PDF |

## How to Use

1. Visit [file2file.me](https://file2file.me)
2. Select your file
3. Choose output format
4. Download converted file
